# Consolidate .bib files into a Single File

Consolidate .bib files into a Single File

## Usage

``` r
Bib(path, bib_lib = FALSE)
```

## Arguments

- path:

  Character string. Project path.

- bib_lib:

  Logical. If `bib_lib = TRUE`, build `bib.bib` from `lib-*` GitHub
  repositories.
