# Install `~/.TinyTex`

Install `~/.TinyTex`

## Usage

``` r
TinyTex(path, bundle = "TinyTeX-2", force = FALSE)
```

## Arguments

- path:

  Character string. Project path.

- bundle:

  The bundle name of TinyTeX (which determines the collection of LaTeX
  packages to install). See
  <https://github.com/rstudio/tinytex-releases#releases> for all
  possible bundles and their meanings.

- force:

  Logical. Force install.
