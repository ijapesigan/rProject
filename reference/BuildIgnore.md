# Build `.Rbuildignore`

Build `.Rbuildignore`

## Usage

``` r
BuildIgnore(path, project, add = NULL)
```

## Arguments

- path:

  Character string. Project path.

- project:

  Character string. Project name.

- add:

  Character vector. Items to add to `.Rbuildignore`.
